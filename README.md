# Sales_pipeline
During my internship, I identified a critical bottleneck in our sales pipeline data: sales representatives frequently forgot to update their "Final Closing Date" when deals stalled. This left 400+ past-due opportunities active in the system.

Tech Stack & Skills
CRM Platform: Zoho CRM (Data Extraction & Operational Alerts)
BI Tool: Microsoft Power BI Desktop
Data Transformation: Power Query (Custom Column handling for Snapshot intervals)
Languages: DAX (Data Analysis Expressions) for conditional KPI tracking

*Note: To protect corporate confidentiality and adhere to strict data privacy policies, all client names, employee names, and exact revenue figures have been completely masked, randomized, or scrubbed prior to modeling. The repository contains a `.pbit` (Power BI Template) file which contains the visual schema and DAX logic with zero underlying data packaged.*
